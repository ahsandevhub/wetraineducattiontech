## ๐ก Design Philosophy

### The 10-70-20 Rule

**10% Yellow (Primary Brand)**

- Call-to-action buttons
- Active navigation items
- Key metrics icons
- Focus states
- Important badges

**70% Neutral Grays (Structure)**

- Backgrounds
- Cards
- Tables
- Body text
- Borders

**20% Accent & Status Colors**

- Success/warning/error indicators
- Chart colors
- Highlights
- Interactive states

### Color Distribution Example

**Page Background:**

- `bg-background` โ `oklch(0.99 0 0)` (Soft white)

**Cards:**

- `bg-card` โ `oklch(1 0 0)` (Pure white)
- `border-border` โ `oklch(0.90 0 0)` (Light gray)

**Primary Actions:**

- `bg-primary` โ `oklch(0.85 0.20 96)` (#facc15)
- `text-primary-foreground` โ `oklch(0.20 0 0)` (Dark text)

**Stat Icons:**

- Container: `bg-primary-100` โ `oklch(0.95 0.06 96)` (Very light yellow)
- Icon: `text-primary-600` โ `oklch(0.72 0.18 96)` (Medium yellow)

**Text Hierarchy:**

- Headlines: `text-foreground` โ `oklch(0.20 0 0)` (Near black)
- Labels: `text-muted-foreground` โ `oklch(0.55 0 0)` (Gray)
